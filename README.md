# Guru's Sweets & Snacks Hero

A premium scroll-driven landing page built with React 19, Vite, Tailwind CSS v4, GSAP, hls.js, and react-router-dom.

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`

## Notes

- The custom headline font is expected at `public/Dirtyline-36daysoftype-2022.woff2`.
- The page uses a black background with white text and a 500vh scroll container.

## Vercel Deployment

- This project is ready for Vercel as a static Vite app.
- The `vercel.json` rewrite keeps React Router refreshes on the client-side app.
